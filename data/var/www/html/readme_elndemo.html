ELN - Electronic (Laboratory) Notebook

Welcome! The Electronic Laboratory Notebook (ELN), designed first to replace the Paper Laboratory Notebook, is also an easy and useful way to keep notes of your daily work whether it's laboratory related or not. It is adapted to everyone: biologist, epidemiologist, statistician, IT developers, ... and is an easy way to share secure information with colleagues. The ELN is based on WordPress. It contains a child theme of Atahualpa and a plugin, IARC Customization, that personalizes WordPress and some plugins used by the ELN. This demo version has been tested with WordPress 4.9.2

— Lucile Alteyrac

Configuration

After the installation, WordPress, the theme and some plugins have been configured.

  1. WordPress
    • General
    • Reading
    • Discussion
    • Permalinks
  2. Theme
    • Activation of Ataiarc
    • Import of settings from file ata-theme-options-for-ELN.txt
  3. Widgets
    • (Left Sidebar) Collapsing Categories, Recent Posts, Last Updated Posts Widget, Recent Comments
    • (Right Sidebar) Category Wise Search, My link order (Management), Text [Subscribe to Posts], Archives, Calendar, User's Favorites
  4. Plugins
    • (Configuration Mandatory) Login Configurator
    • (Configuration Recommended) IARC Customization
    • (Configuration Recommended) TinyMCE Advanced: import of settings from file tinymce-settings.txt
    • (Configuration Recommended) Post Notification
    • (Configuration Optional) WP Favorite Posts
    • (Configuration Optional) Print
    • (Configuration Optional) PageNavi

Content

Content has been added to this demo:

  1. Categories (the WP categories are named "Notebooks" in the ELN):
    • Biobank
      • Shipment
      • Storage
    • Laboratories
      • Mass spectrometry
      • PCR
  2. Role Groups:
    • Biobank Group
    • Lab Group
  3. Users:
    • elnadmin: WordPress Administrator
    • elnsupervisor: Read-only user, with an access to all notebooks
    • [Biobank Group] biobankadm: read+write access to all Biobank Group's notebooks
    • [Lab Group] labtech: Lab Group's notebooks read-only user
    • [Lab Group] mstech: Lab Group's notebooks read-only user, can write in Mass Spectrometry notebook
    • [Lab Group] pcrtech: Lab Group's notebooks read-only user, can write in PCR notebook
  4. Pages:
    • Home Page
    • User Guide
  5. Posts: Some posts have been created, so the notebooks will be listed in the "Notebooks" widget
  6. Links: Creation of the links used in the "Management" widget

The configuration of permissions for Groups and Users is done with the plugin Role Scoper.
