This widget plugin displays the list of the last updated/modified posts in your sidebar.



== Description ==

THIS PLUGIN IS NOT SUPPORTED AT THE MOMENT

When you update/modify one of your posts on your Wordpress blog and you want your visitors to know about it, this is what you need. This widget plugin create a sidebar box with a list of posts you've recently updated/modified. 
You can choose to display the date of the update beside the page title in the format you prefer. 

Have fun!



== Installation ==

To install the plugin, please follow these steps:

1. Download the zipped plugin file to your local machine.
2. Unzip the file.
3. Upload the *'last-updated-posts-widget'* folder to the *'/wp-content/plugins/'* directory on your webserver.
4. Activate the plugin through the 'Plugins' menu in WordPress.
5. Go to 'Widgets' section under 'Appearance' menu in WordPress and add the *'Last Updated Posts Widget'* in your sidebar.
6. Open the widget options and...
   Write the number of posts to show (leave empty to show all posts).
   Checkbox to show data and choose data format.
   Insert a widget title to fit your design.
